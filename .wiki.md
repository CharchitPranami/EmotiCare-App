# Project Summary
EmotiCare is a locally-run Gradio web application designed to support emotional well-being by detecting user moods through text and audio inputs. Utilizing Google Gemini 1.5 Flash, it provides therapy-style coaching, journaling prompts, and crisis safety checks, all aimed at facilitating emotional management and providing timely support for users.

# Project Module Description
- **Mood Detection**: Analyzes user input to identify emotional states (e.g., happy, sad, anxious).
- **Therapy Coach**: Offers empathetic responses and actionable steps based on detected moods.
- **Crisis Safety Checks**: Flags high-risk language and displays emergency resources if necessary.
- **Journaling**: Maintains a history of user entries and suggests prompts for deeper exploration.
- **UI/UX Enhancements**: Updated interface with a calming color scheme and improved layout for better user interaction.

# Usage
1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Get Gemini API Key**:
   - Obtain your API key from Google AI Studio.
3. **Set API Key**:
   - Create a `.env` file in the project directory.
   - Add the line: `GEMINI_API_KEY=your_actual_api_key_here`.
4. **Run the App**:
   ```bash
   python app.py
   ```
